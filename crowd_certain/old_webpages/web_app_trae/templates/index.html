Crowd-Certain Help

About Crowd-Certain

Crowd-Certain is a framework for evaluating and comparing different crowd-sourced label aggregation techniques. It focuses on understanding the relationship between worker quality and the weights assigned by different aggregation methods.

How to Use This Dashboard

  1. Select Datasets: Choose one or more datasets to run simulations on.
  2. Configure Workers: Set the minimum and maximum number of workers and their quality range.
  3. Choose Techniques: Select uncertainty and consistency techniques to evaluate.
  4. Run Simulation: Click the "Run Simulation" button to start the simulation.
  5. Explore Results: View performance metrics and weight-strength relationships in the results panel.

Understanding the Results

Performance Metrics: Shows accuracy, F1 score, and other metrics for both proposed and benchmark methods.

Weight-Strength Relation: Visualizes how worker quality (strength) correlates with the weights assigned by different aggregation methods.
