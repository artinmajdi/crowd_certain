About Crowd-Certain

Key Features

  • Uncertainty Quantification: Estimate the uncertainty in aggregated labels using multiple techniques
  • Consistency Measurement: Evaluate the consistency of crowd workers' responses
  • Worker Quality Assessment: Analyze individual worker performance and reliability
  • Benchmark Comparisons: Compare against established crowdsourcing algorithms
  • Dataset Support: Works with various datasets for comprehensive evaluation

Benchmark Methods

Crowd-Certain compares its proposed methods against these established benchmarks:

  • Majority Vote: Simple voting mechanism where each worker has equal weight
  • MACE: Multi-Annotator Competence Estimation
  • Dawid-Skene: EM-based algorithm that models worker confusion matrices
  • GLAD: Generative model of Labels, Abilities, and Difficulties
  • ZeroBasedSkill: Estimates worker skill levels relative to a zero baseline

Dataset Structure

Crowd-Certain works with datasets that contain:

  • Task identifiers
  • Worker identifiers
  • Worker labels/annotations
  • Ground truth labels (for evaluation)
